# React-Boilerplate
This is a react-boilerplate created for leaning

## Setup
npm install
npm run build
npm start

## Goals
- [x] Jest
- [x] Enzyme
- [x] React-emotion
- [x] ~webpack-dev-server~ webpack-serve
- [x] eslint rules and prettier rules
- [x] Hot reload client (webpack-serve + react-hot-loader)
- [ ] Redux
- [ ] React router 
- [ ] code splitting by route
- [ ] SSR-ready
- [ ] Hot reload server
